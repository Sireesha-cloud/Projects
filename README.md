# Restaurant Sales & Analytics Data Platform

An end-to-end restaurant data engineering and analytics project using Python, Pandas, SQLite, and Excel POS data.

## Features
- POS data ingestion from Excel
- ETL pipeline (Extract, Transform, Load)
- Data warehouse (SQLite)
- Sales & revenue analytics
- Customer analytics
- Demand forecasting
- Business intelligence dashboards

## Tech Stack
Python, Pandas, SQLite, SQL, Matplotlib, Google Colab

## Dataset
Restaurant POS data located in:
data/raw_orders.xlsx

## How to Run
1. Open Restaurant_Analytics.ipynb in Google Colab  
2. Upload data/raw_orders.xlsx  
3. Run all cells  
4. View analytics, forecasts, and reports
